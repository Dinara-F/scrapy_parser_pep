# Проект парсинга scrapy pep

## Описание

Проект SCRAPY_PAESER_PEP это приложение, которое асинхронно парсит всякую новую инфу про питон и сохраняет ее в файлы csv иногда с помощью FEEDS, а иногда с помощью pipelines.
